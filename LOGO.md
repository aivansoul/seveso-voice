# Logo Banana Navy

`banana-navy-logo.svg` est une **reconstitution vectorielle** de la marque (voile + coque banane, monochrome sombre) pour que le dashboard s'affiche correctement immédiatement.

**Pour la version finale :** déposez le PNG officiel haute résolution ici sous le nom **`banana-navy-logo.png`** (fond transparent de préférence). Le dashboard préfère le PNG s'il existe, sinon il utilise le SVG.

Charte appliquée dans le dashboard :
- fond clair gris-blanc ;
- aucune émoji ;
- icônes vectorielles uniquement (Lucide).
